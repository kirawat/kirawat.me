Security

Password Hashing

Appwrite protects password by using the Argon2 password-hashing algorithm … combines with the use of techniques such as salting, adjustable work factors, and memory hardness to securely handle passwords.1

Appwrite is certified with the European General Data Protection Regulation (GDPR) standards.2

Appwrite is also actively working towards compliance with SOC 2 standards and HIPPA, which will be announced at a later date.2
