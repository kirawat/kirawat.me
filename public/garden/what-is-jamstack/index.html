What is Jamstack?

  • Programming/Web
Planted  21 Dec 2023 13:01:00 PM (+07:00)
Last tended  21 Dec 2023 13:10:35 PM (+07:00)
DigitalOcean

The “JAM” in Jamstack stands for JavaScript, APIs, and Markdown. Jamstack applications use a combination of client-side JavaScript, third-party APIs, and static HTML to create a website that has the same robust functionality as a full-stack website but without the usual backend infrastructure to handle and process requests.1
